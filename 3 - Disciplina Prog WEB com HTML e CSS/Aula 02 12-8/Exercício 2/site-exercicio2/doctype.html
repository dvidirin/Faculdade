Declaração <!DOCTYPE html>

- Deve ser a primeira linha do documento html, antes mesmo da tag <html>.

- Não é uma tag html, e também não é sensível ao caso.

- As tags HTML normalmente vêm em pares como <p> e </p>.

- É uma instrução ao browser sobre qual a versão de HTML a página foi escrita.

- A declaração <! DOCTYPE html> declara que o documento está escrito em HTML5.
