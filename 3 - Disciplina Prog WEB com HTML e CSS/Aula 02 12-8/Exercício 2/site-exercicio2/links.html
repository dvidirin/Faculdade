Links

- Para criar um hiperlink em uma página html utilizamos a tag:

<a href = "insira o link aqui entre aspas"> Essa é a frase que deve ser clicada. </a>

Exemplos:

- <a href="https://pt.wikipedia.org/wiki/Wikip%C3%A9dia:P%C3%A1gina_principal"> Wikipédia </a>

é a declaração de um link para a página da Wikipédia a seguir:

- Wikipédia

- <a href="index.html"> Volta </a>

é a declaração do link de volta dessa página, que está local no meu hd.

- Volta
