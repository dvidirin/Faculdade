O que são Tags?

- As tags HTML são nomes de elementos rodeados por colchetes angulares (< >).

- < tagname > aqui vai o conteúdo... < /tagname >

- As tags HTML normalmente vêm em pares como <p> e </p>.

- A primeira tags em um par é o início e a segunda tags é o final.

- O final da tags é escrito como o início da tags, mas com uma barra inserida antes do noma da tags.

- As tags HTML não são case-sensitive, ou seja, não diferenciam letras maiúsculas de minúsculas (<p> e <P> são iguais).

- O padrão HTML5 não requer tags em minúsculas, mas o W3C recomenda minúsculas em HTML e exige minúsculas para tipos de documentos mais rígidos, como XHTML.
